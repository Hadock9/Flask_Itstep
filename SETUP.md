# Інструкція для налаштування ChatGPT інтеграції

## Крок 1: Встановлення бібліотеки OpenAI

```bash
pip install openai
```

Або якщо використовуєте віртуальне середовище:

```bash
source venv/bin/activate  # для macOS/Linux
# або
venv\Scripts\activate  # для Windows

pip install openai
```

## Крок 2: Отримання API ключа

1. Перейдіть на https://platform.openai.com/api-keys
2. Увійдіть або зареєструйтеся
3. Створіть новий API ключ
4. Скопіюйте ключ (він починається з `sk-...`)

## Крок 3: Встановлення API ключа

### macOS/Linux (zsh/bash):

**Тимчасово (тільки для поточної сесії терміналу):**

```bash
export OPENAI_API_KEY="sk-ваш-ключ-тут"
```

**Постійно (додати в ~/.zshrc або ~/.bashrc):**

```bash
echo 'export OPENAI_API_KEY="sk-ваш-ключ-тут"' >> ~/.zshrc
source ~/.zshrc
```

### Windows (PowerShell):

```powershell
$env:OPENAI_API_KEY="sk-ваш-ключ-тут"
```

### Windows (Command Prompt):

```cmd
set OPENAI_API_KEY=sk-ваш-ключ-тут
```

## Крок 4: Запуск сервера

```bash
python app.py
```

Або:

```bash
flask run
```

## Перевірка

1. Відкрийте http://localhost:5000/chat
2. Введіть повідомлення
3. Якщо все налаштовано правильно, ви отримаєте відповідь від GPT

## Діагностика помилок

Якщо чат не працює, перевірте:

1. **"OpenAI бібліотека не встановлена"**

   - Виконайте: `pip install openai`

2. **"OPENAI_API_KEY не встановлено"**

   - Перевірте, чи встановлено змінну середовища: `echo $OPENAI_API_KEY`
   - Якщо порожньо, встановіть ключ (див. Крок 3)

3. **"Помилка звернення до OpenAI"**
   - Перевірте правильність API ключа
   - Перевірте, чи є кошти на рахунку OpenAI
   - Перевірте деталі помилки в консолі браузера (F12)

## Безпека

⚠️ **НІКОЛИ не додавайте API ключ безпосередньо в код!**

- Використовуйте змінні середовища
- Не комітьте ключі в Git
- Додайте `.env` в `.gitignore` якщо використовуєте файл змінних середовища

## Безкоштовна альтернатива: Groq API

Groq надає безкоштовний доступ до швидких LLM моделей (Llama 3.1).

### Крок 1: Отримайте безкоштовний API ключ

1. Перейдіть на https://console.groq.com/keys
2. Увійдіть через Google/GitHub
3. Створіть новий API ключ
4. Скопіюйте ключ

### Крок 2: Додайте ключ в .env файл

```bash
GROQ_API_KEY=ваш-ключ-тут
```

Або встановіть змінну середовища:

```bash
export GROQ_API_KEY="ваш-ключ-тут"
```

### Переваги Groq:

- ✅ Повністю безкоштовний
- ✅ Дуже швидкий (використовує спеціальне обладнання)
- ✅ Висока якість відповідей (Llama 3.1)
- ✅ Великі ліміти запитів

Код автоматично використовує Groq, якщо він налаштований, інакше пробує OpenAI.
